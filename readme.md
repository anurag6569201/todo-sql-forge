## Simple To-do app that uses MySQL as database
Is project ka use case me kuch unique nhi hai. But yes, using MySQL as database is the main thing here.

## Functionalities:

1. Add a new task.
2. Delete an existing task.
3. Edit/Update an existing task.
4. Check is completed/non-completed.

<ol>

## To contribute:

1. Fork and clone the repo.
2. Add the environment variables using a `.env` file to the root backend directory.

```shell
DB_NAME=******************
DB_USER=******************
DB_PASSWORD=******************
DB_HOST=******************
DB_PORT=******************
```
3. Make sure you've mysql installed in your system. 
4. To run in dev mode:
```shell
cd frontend
npm install
npm run dev
```

```shell
cd backend
npm install
npx nodemon server.js
```
5. Create a new branch.
6. Stage and commit your changes.
7. Open a pull request.

## Deployment Scope

1. Frontend to be hosted on **vercel.com**.
2. Database to be hosted on **railway.app**.
3. Backend to be hosted on **render.com**.
